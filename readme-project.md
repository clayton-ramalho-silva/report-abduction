## Sobre o Projeto - Report Abduction

O Projeto REPORT ABDUCTION faz parte de um projeto maior. Chamado "Use a cabeça PHP & MySQL com Laravel". Que consiste em replicar os exemplos do Livro Use a cabeça - PHP & MySQL, originalmente executados com PHP 4 puro, utilizando Laravel na versão 8. 

O intuito é puramente didádico para o aprofundamento do estudo no Framework Laravel·
